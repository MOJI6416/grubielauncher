# О проекте

**Grubie Launcher** — современный лаунчер для Minecraft с упором на удобство, социальные функции и управление сборками. Он помогает устанавливать версии, управлять модами и мирами, делиться сборками и поднимать серверы прямо из лаунчера.

# Возможности

- Установка и запуск **официальных версий Minecraft**.
- Поддержка **Forge**, **Fabric**, **NeoForge** и **Quilt**.
- Автоматическая загрузка Java и зависимостей.
- Поддержка нескольких аккаунтов: **Microsoft/Mojang**, **Ely.by**, **Discord** и оффлайн.
- Удобное управление версиями, модами и мирами.
- Интеграция с **CurseForge** и **Modrinth**.
- Система скинов: Microsoft, Ely.by и **собственные** скины.
- Система друзей.
- Достижения и внутренняя статистика игрока.
- Система публикации сборок: игроки вашей сборки получают обновления автоматически.
- Функция **«Открыть доступ к миру»** через интернет с режимами **public** и **friends**.
- Создание и управление сервером прямо из лаунчера. _Поддерживаемые ядра серверов: Vanilla, Spigot, Bukkit, Paper, Purpur, Forge, NeoForge, Fabric, Quilt._

# Установка и запуск

1. Скачайте последнюю версию из [Releases](https://github.com/MOJI6416/grubielauncher/releases).
2. Запустите `grubie-launcher`.

_(Поддерживаемые платформы: Windows, Linux)_
